Detailed Description

template<class Key, class T, class Hash = std::hash<Key>, class KeyEqual = std::equal_to<Key>, class HopInfosType = std::uint64_t>
class hopscotch_map< Key, T, Hash, KeyEqual, HopInfosType >

Implementation of a hash map using the hopscotch hashing algorithm.

The size of the neighborhood (H or MAX_HOPS) is defined by the template parameter HopInfosType. H equals to the number of bits in HopInfosType - 2 reserved bits (ex. for HopInfosType = uint64, H will be 62 (64-2)).
